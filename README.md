# 📑 lista-avaliacao-zero

*Este repositório contém a resolução de uma lista de atividades práticas desenvolvidas em Linguagem C.* 

---

## Descrição das questões
  - `q1.c`: Solicita um número inteiro positivo e imprime a sequência de 1 até o valor informado.
  - `q2.c`: Calcula e exibe o fatorial de um número inteiro positivo fornecido pelo usuário.
  - `q3.c`: Lê um array de 5 números inteiros e calcula a soma total de seus elementos.
  - `q4.c`: Implementa um conversor de temperatura interativo entre Celsius e Fahrenheit com menu de opções.
  - `q5.c`: Recebe uma string e conta o número total de caracteres, desconsiderando a quebra de linha.
  - `q6.c`: Verifica se um número inteiro maior que 1 é um número primo ou não.

---

## Como executar
Para compilar e rodar os exercícios, é necessário ter o **GCC** (ou outro compilador C) instalado em sua máquina.

### 1️⃣ Passo: compilação
Abra o terminal na pasta do projeto e digite:

`gcc q1.c -o programa`

*(substitua `q1.c` pelo arquivo que deseja testar)*

### 2️⃣ Passo: Execução
Execute o arquivo gerado de acordo com seu sistema operacional:

- **Windows**:

  `./programa.exe`
  
- **Linux/Mac**:

  `./programa`
